# Freelance Job Parser & Auto-Responder Bot

Автоматический бот для парсинга заданий с фриланс бирж, отправки персонализированных предложений и уведомлений в Telegram.

## 🚀 Возможности

- ✅ Парсинг заданий с FL.ru и Freelance.ru
- ✅ Автоматическая отправка персонализированных предложений
- ✅ Уведомления в Telegram о новых заданиях
- ✅ Фильтрация по ключевым словам и бюджету
- ✅ Отслеживание уже обработанных заданий
- ✅ Настраиваемые интервалы проверки
- ✅ Логирование всех операций

## 📋 Требования

- Python 3.7+
- Telegram бот (токен)
- Аккаунты на фриланс биржах

## 🔧 Установка

### 1. Клонируйте или скачайте проект

```bash
cd c:\Users\Andrey\Desktop\freelance
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Создайте Telegram бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен бота
4. Получите свой Chat ID:
   - Найдите [@userinfobot](https://t.me/userinfobot)
   - Отправьте ему любое сообщение
   - Скопируйте ваш ID

### 4. Настройте конфигурацию

Откройте файл `config.json` и заполните необходимые данные:

```json
{
  "telegram_bot_token": "123456789:ABCdefGHIjklMNOpqrsTUVwxyz",
  "telegram_chat_id": "123456789",
  
  "enable_fl_ru": true,
  "fl_ru_login": "ваш_логин",
  "fl_ru_password": "ваш_пароль",
  
  "auto_send_proposals": false,
  "check_interval": 300,
  
  "keywords": ["python", "парсинг", "бот"],
  "exclude_keywords": ["дешево", "бесплатно"],
  "min_budget": 5000
}
```

## ⚙️ Настройка

### Основные параметры

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| `telegram_bot_token` | Токен вашего Telegram бота | - |
| `telegram_chat_id` | Ваш Telegram Chat ID | - |
| `check_interval` | Интервал проверки новых заданий (секунды) | 300 |
| `auto_send_proposals` | Автоматически отправлять предложения | false |
| `delay_between_proposals` | Задержка между отправками (секунды) | 10 |

### Фильтры

- **keywords** - список ключевых слов для поиска (задание должно содержать хотя бы одно)
- **exclude_keywords** - список исключающих слов (задание не должно содержать ни одного)
- **min_budget** - минимальный бюджет задания (в рублях)

### Настройка FL.ru

```json
{
  "enable_fl_ru": true,
  "fl_ru_login": "ваш_логин",
  "fl_ru_password": "ваш_пароль",
  "fl_ru_categories": ["", "programming", "web-programming"]
}
```

### Настройка Freelance.ru

```json
{
  "enable_freelance_ru": true,
  "freelance_ru_login": "ваш_логин",
  "freelance_ru_password": "ваш_пароль"
}
```

### Шаблон предложения

Вы можете настроить шаблон предложения в `config.json`:

```json
{
  "proposal_template": "Здравствуйте!\n\nЗаинтересовало ваше задание \"{title}\".\n\nИмею опыт работы с подобными проектами.\n\nС уважением"
}
```

Доступные переменные:
- `{title}` - название задания
- `{description}` - описание задания
- `{budget}` - бюджет задания
- `{client_name}` - имя заказчика

## ⚠️ Важные замечания

### Безопасность

1. **Не публикуйте config.json** с вашими реальными данными
2. Храните токены и пароли в безопасности
3. Используйте `.gitignore` для исключения конфиденциальных файлов

### Ограничения

1. **Структура сайтов может меняться** - парсеры могут требовать обновления
2. **Лимиты бирж** - соблюдайте правила использования API и не злоупотребляйте частотой запросов
3. **Капча** - некоторые биржи могут использовать защиту от ботов

### Рекомендации

1. **Тестируйте с `auto_send_proposals: false`** - сначала проверьте, что бот корректно находит задания
2. **Настройте фильтры** - чтобы получать только релевантные задания
3. **Персонализируйте шаблон** - уникальные предложения повышают шансы на отклик
4. **Проверяйте логи** - регулярно просматривайте `freelance_parser.log`

## 🔄 Обновление парсеров

Если структура сайта изменилась, вам нужно обновить селекторы в классах парсеров:

### Для FL.ru (в файле `freelance_parser.py`):

```python
# Найдите метод parse_jobs в классе FL_RuParser
job_items = soup.find_all('div', class_='project-item')  # Обновите селектор
```

### Для Freelance.ru:

```python
# Найдите метод parse_jobs в классе FreelanceRuParser
job_items = soup.find_all('div', class_='task-item')  # Обновите селектор
```
