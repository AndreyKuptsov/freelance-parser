# 🚀 Быстрый старт

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Создание Telegram бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя и username для бота
4. Скопируйте полученный **токен** (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 3: Получение Chat ID

1. Найдите в Telegram **@userinfobot**
2. Отправьте ему любое сообщение
3. Скопируйте ваш **ID** (например: `123456789`)

## Шаг 4: Настройка config.json

Откройте файл `config.json` и заполните:

```json
{
  "telegram_bot_token": "ВАШ_ТОКЕН_БОТА",
  "telegram_chat_id": "ВАШ_CHAT_ID",
  
  "enable_fl_ru": true,
  "fl_ru_login": "ваш_логин_на_fl_ru",
  "fl_ru_password": "ваш_пароль",
  
  "auto_send_proposals": false,
  "keywords": ["python", "парсинг", "бот"]
}
```

### ⚠️ Важно:

- **auto_send_proposals: false** - бот только уведомляет, не откликается автоматически
- **auto_send_proposals: true** - бот автоматически откликается на задания (используйте осторожно!)

## Шаг 5: Запуск

```bash
python freelance_parser.py
```

## Что происходит?

1. ✅ Бот подключается к FL.ru
2. ✅ Каждые 5 минут проверяет новые задания
3. ✅ Фильтрует по вашим ключевым словам
4. ✅ Отправляет уведомления в Telegram
5. ✅ Сохраняет обработанные задания (не дублирует)

## Пример уведомления в Telegram

```
🆕 Новое задание!

📋 Название: Разработка Telegram бота
💰 Бюджет: 15000 ₽
🌐 Источник: FL.ru
🔗 Ссылка: https://www.fl.ru/projects/...

ℹ️ Уведомление (автоотправка выключена)

Описание:
Требуется разработать бота для автоматизации...
```

## Настройка фильтров

### Ключевые слова (keywords)
Задание должно содержать **хотя бы одно** из этих слов:

```json
"keywords": ["python", "парсинг", "бот", "telegram", "django"]
```

### Исключающие слова (exclude_keywords)
Задание **не должно** содержать эти слова:

```json
"exclude_keywords": ["дешево", "бесплатно", "копирайтинг"]
```

### Минимальный бюджет

```json
"min_budget": 5000
```

## Остановка бота

Нажмите `Ctrl+C` в терминале

## Логи

Все действия записываются в файл `freelance_parser.log`

## Проблемы?

### Бот не находит задания
- Проверьте файл `debug_fl_ru.html` - там сохранена страница для анализа
- Возможно, структура сайта изменилась

### Ошибка авторизации
- Проверьте логин и пароль в `config.json`
- Убедитесь, что аккаунт активен

### Не приходят уведомления в Telegram
- Проверьте токен бота и chat_id
- Убедитесь, что вы написали боту хотя бы одно сообщение

## Следующие шаги

1. **Протестируйте** с `auto_send_proposals: false`
2. **Настройте фильтры** под свои навыки
3. **Персонализируйте** шаблон предложения
4. **Включите автоотправку** когда будете готовы

---

💡 **Совет**: Начните с узких фильтров и постепенно расширяйте их, чтобы получать только релевантные задания.
